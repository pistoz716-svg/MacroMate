# MacroMate

MacroMate is a web-based macro calculator, meal generator, saved meal planner, and grocery list tool built with HTML, CSS, and JavaScript.

## Live Demo

```txt
https://pistoz716-svg.github.io/MacroMate/
```

## Features

- Weight goal calculator
- Daily macro calculator
- Calorie, protein, carb, and fat targets
- Meal logger
- Breakfast, lunch, snack, and dinner generator
- Full-day meal plan generator
- Recipe cards with ingredients, directions, storage, and substitutions
- Saved daily meal plan
- Grocery list generator
- Macro coverage comparison
- Local storage persistence
- Responsive mobile-friendly interface

## Tech Stack

- HTML5
- CSS3
- JavaScript
- LocalStorage
- GitHub Pages

## How To Run Locally

Open the project folder and launch `index.html` with a local server.

Recommended with VS Code:

1. Install the Live Server extension.
2. Open the `MacroMate` folder.
3. Right-click `index.html`.
4. Select **Open with Live Server**.
