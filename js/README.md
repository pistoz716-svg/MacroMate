# MacroMate

MacroMate is a simple macro counter, macro adjuster, and meal planner.

## Features

- Calculate calories
- Calculate protein, carbs, and fat
- Log meals
- Track remaining macros
- Generate a simple meal plan
- Runs with plain HTML, CSS, and JavaScript

## How to Run

Open `index.html` in your browser.

Or run locally:

```bash
npm install
npm start
