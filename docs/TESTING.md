# MacroMate Testing Plan

## Manual Testing Checklist

### 1. Macro Calculator

- Enter valid current weight.
- Enter valid goal weight.
- Select pounds.
- Select kilograms.
- Select each goal type.
- Select each activity level.
- Confirm calories, protein, carbs, and fat update.

### 2. Meal Logger

- Select a food.
- Enter 1 serving.
- Add food.
- Add multiple foods.
- Add decimal servings such as 0.5 or 1.25.

### 3. Meal Generator

- Generate breakfast.
- Generate lunch.
- Generate snack.
- Generate dinner.
- Generate full day.

### 4. Saved Meals

- Generate breakfast and save.
- Generate lunch and save.
- Generate snack and save.
- Generate dinner and save.
- Refresh browser and confirm saved meals remain.

### 5. Grocery List

- Save one meal.
- Generate grocery list.
- Save full day.
- Generate grocery list again.
- Clear saved meals.

### 6. Responsive Design

Test:

- Desktop
- Laptop
- Tablet
- Mobile

## Browser Testing

- Chrome
- Edge
- Firefox
- Safari if available

## Known Risks

- LocalStorage does not sync across devices.
- Food data is approximate.
- Recipe scaling is estimated.
- No authentication yet.
