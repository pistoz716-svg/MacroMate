# Fredrick Johnson IV — Software Project Portfolio

This portfolio showcases practical software applications built around real-world problems, product thinking, and full software development lifecycle practices.

## Featured Projects

### MacroMate
A nutrition and meal planning platform that helps users calculate macros, generate meals, save meal plans, and create grocery lists.

**Focus Areas:**  
Frontend development, data modeling, local storage, meal planning logic, grocery generation, responsive UI, GitHub Pages deployment.

---

### Alkaline Food Scanner
A Dr. Sebi-inspired food scanner and swap engine that helps users identify alkaline-friendly foods and find cleaner alternatives.

**Focus Areas:**  
Food classification, scanner workflow, health/wellness UX, data filtering, swap recommendations.

---

### Estimatic
An AI-powered estimating tool that helps users generate project, repair, and material estimates quickly.

**Focus Areas:**  
AI-assisted workflows, forms, estimate generation, material breakdowns, productivity tooling.

---

### DANI
A DIY assistant that helps users diagnose projects, calculate materials, estimate costs, and follow repair guidance.

**Focus Areas:**  
AI assistant design, repair guidance, material planning, step-by-step workflows, practical automation.
