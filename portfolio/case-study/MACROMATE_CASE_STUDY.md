# MacroMate Case Study

## Overview

MacroMate is a nutrition-focused web application that helps users calculate macros, generate meals, save meal plans, and build grocery lists.

## Problem

Many macro calculators tell users their calorie and macro goals but do not help them translate those numbers into real meals and grocery lists.

## Solution

MacroMate connects macro targets to actionable meal planning by generating recipes, saving meals, and creating grocery lists based on the user’s goals.

## Core Features

- Macro calculator
- Weight goal calculator
- Meal logger
- Recipe generator
- Saved daily meal plan
- Grocery list generator
- Dashboard navigation
- LocalStorage persistence
- GitHub Pages deployment
